					===================================================
							      Inner OR Nested Loops--Most Imp
					===================================================
=>The Process of Defining One Loop inside of another Loop is called Inner OR Nested Loop.
=>The Execution Process of Inner OR Nested Loops is that "For Every Value of Outer Loop, Inner Loop Executes for finite Number of times until Inner Loop Test Condition False".
=>In Python , we can use Inner OR Nested Loops in 4 ways. They are
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach1:   for loop in for loop
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:   for varname-1 in Iterable-object-1:  # outer loop
			------------------------------------------------
			------------------------------------------------
			for varname-2 in Iterable-object-2: # Inner loop
				------------------------------------------------
				------------------------------------------------
			else: # Inner loop else part
				---------------------------------------
				---------------------------------------
	        else: # outer loop else part
			----------------------------------
			-----------------------------------
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach2:   while loop in while loop
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:       ------------------------------------------------
		    while(Test Cond1):  # outer loop
				-----------------------------------------
				-----------------------------------------
				while(Test Cond2):  #inner loop
					-----------------------------------
					-----------------------------------
				else:  # inner loop else part
					-----------------------------------
		    else:  # outer loop else part
				--------------------------------------------
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach3:   for loop in while loop
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:       ------------------------------------------------
		    while(Test Cond):  # outer loop
				-----------------------------------------
				-----------------------------------------
				for varname in Iterable-object: # Inner loop
					------------------------------------------------
					------------------------------------------------
				else:  # inner loop else part
					-----------------------------------
		    else:  # outer loop else part
				--------------------------------------------
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach4:   while loop in for loop
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:   for varname in Iterable-object:  # outer loop
			------------------------------------------------
			------------------------------------------------
			while(Test Cond):  #inner loop
					-----------------------------------
					-----------------------------------
			else: # Inner loop else part
				---------------------------------------
				---------------------------------------
		else: # outer loop else part
			----------------------------------
			-----------------------------------
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
