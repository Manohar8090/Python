				=====================================================
								4. range
				=====================================================
=>Here 'range' is na Pre-Defined Class and Treated as Sequence  Data Type.
=>The Purpose of 'range' data type is that "To store Sequence of Numerical Integer values by maintaining equal Interval of Step either In Forward Direction OR Backward Direction".
=>On the Object of range, we can perform Both Indexing and Slicing Operations.
=>An Object of range belongs to Immutable bcoz It does  not support Item Assignment.
=>An Object of range maintains Insertion Order.
=>To store or organize the Sequence of Numerical Integer values by maintaining equal Interval of Step either In Forward Direction OR Backward Direction we use 3 Pre-Defined functions of range data type. They are
----------------------------------------------------------------------------------------------------------------------------------------------------
Syntax-1:  varname=range(Value)
----------------------------------------------------------------------------------------------------------------------------------------------------
=>This Syntax generates range of values from 0 to Value-1 with default Step +1 (Forward Direction)
----------------------------------------------------------------------------------------------------------------------------------------------------
Syntax-2:  varname=range(Start,Stop)
----------------------------------------------------------------------------------------------------------------------------------------------------
=>This Syntax generates range of values from Start to Stop-1 with default Step +1 (Forward Direction)
----------------------------------------------------------------------------------------------------------------------------------------------------
Syntax-3:  varname=range(Start,Stop,Step)
----------------------------------------------------------------------------------------------------------------------------------------------------
=>This Syntax generates range of values from Start to Stop-1  either in FORWARD OR BACKWARD Direction with Programmer Defined Step.
----------------------------------------------------------------------------------------------------------------------------------------------------
Examples
----------------------------------------------------------------------------------------------------------------------------------------------------
>>> r=range(6)
>>> print(r,type(r))------------------------range(0, 6) <class 'range'>
