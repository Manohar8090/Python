				=======================================================
						Base Converstion Techniques
				=======================================================
=>The Purpose of Base Converstion Techniques is that "To convert One Base Value to another Base Value".
=>In Python Programming, we have 3 Base Convertion Techniques. They are

					1. bin()
					2. oct()
					3. hex()				
									
--------------------------------------------------------------------------------------------------------------------------------------------------------------
1. bin()
--------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:  varname=bin(Decimal / Octal / Hexa)
=>This Function is used for Converting Decimal OR Octal OR Hexa Decimal value into Binary Value.
----------------------------
Examples
----------------------------
>>> a=15
>>> print(a,type(a))-------------------15 <class 'int'>
>>> bv=bin(a)
>>> print(bv)----------------------------0b1111
>>> type(bv)-----------------------------<class 'str'>
--------------------------
>>> a=0o17
>>> bv=bin(a)
>>> print(bv)-------------------------0b1111
------------------------------
>>> a=0xF
>>> bv=bin(a)
>>> print(bv)--------------------------0b1111
--------------------------------------------------------------------------------------------------------------------------------------------------------------
2. oct()
--------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:   varname=oct(Decimal / Binary / Hexa)
=>This Function is used for Converting Decimal, Binary and Hex Decimal values into Octal value.
----------------------------
Examples
----------------------------
>>> a=15
>>> ov=oct(a)
>>> print(ov,type(ov))---------------0o17 <class 'str'>
------------------------------
>>> a=0b1010
>>> ov=oct(a)
>>> print(ov)-----------------0o12
>>> ov=oct(0b1111)
>>> print(ov)-----------------0o17
------------------------------
>>> a=0xF
>>> ov=oct(a)
>>> print(ov)--------------0o17
>>> ov=oct(0xA)
>>> print(ov)------------0o12
--------------------------------------------------------------------------------------------------------------------------------------------------------------
3. hex()
--------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:   varname=hex(Decimal / Binary / Octal)
=>This Function is used for Converting Decimal, Binary and Octal Values in to Hex Decimal values. 
-----------------------
Examples
-----------------------
>>> a=10
>>> hv=hex(a)
>>> print(hv)------------0xa
>>> hv=hex(15)
>>> print(hv)---------------0xf
>>> hv=hex(64206)
>>> print(hv)----------------0xface
>>> hv=hex(3054)
>>> print(hv)---------------0xbee
----------------------------
>>> a=0b1110
>>> hv=hex(a)
>>> print(hv)---------------0xe
>>> hv=hex(0b1111)
>>> print(hv)--------------0xf
-------------------------------
>>> a=0o17
>>> hv=hex(a)
>>> print(hv)-------------0xf
>>> a=0o12
>>> hv=hex(a)
>>> print(hv)--------------0xa
--------------------------------------------------------------------------------------------------------------------------------------------------------------
						Base Conversion Calculator
--------------------------------------------------------------------------------------------------------------------------------------------------------------
				I.   Decimal to Binary------------bin()
				     Decimal to Octal--------------oct()
				     Decimal to Hexa--------------hex()

				II. Binary to Decimal------------Automatically
				     Binary to octal----------------oct()
				     Binary to Hexa----------------hex()

				III. Octal to Decimal-------------Automatically
				      Octal to Binary---------------bin()
				      Octal to Hexa-----------------hex()

				IV. Hexa to Decimal--------------Automatically
				      Hexa  to Binary---------------bin()
				      Hexa to Octal-----------------oct()
