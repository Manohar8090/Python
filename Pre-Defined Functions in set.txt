=>On set Object, we can perform Various Operations by using Pre-Defined Functions in set object. They are
---------------------------------------------------------------------------------------------------------------------------------------------------------
1. add()
---------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax: SetObj.add(Value)
=>This Function is used for adding the value to set object.
---------------------------
Examples
---------------------------
>>> s1={10,"RS","Python"}
>>> print(s1,type(s1),id(s1))
{'RS', 10, 'Python'} <class 'set'> 2378857044384
>>> s1.add("NL")
>>> print(s1,type(s1),id(s1))
{'RS', 10, 'Python', 'NL'} <class 'set'> 2378857044384
>>> s1.add(34.56)
>>> print(s1,type(s1),id(s1))
{'RS', 34.56, 10, 'NL', 'Python'} <class 'set'> 2378857044384
----------------------
>>> s1=set()
>>> print(s1,type(s1),id(s1))
set() <class 'set'> 2378889695232
>>> s1.add(100)
>>> s1.add("Python")
>>> s1.add(3.4)
>>> print(s1,type(s1),id(s1))
{3.4, 'Python', 100} <class 'set'> 2378889695232

2. remove()
---------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:   setobj.remove(Value)
=>This Function removes the  Specified Value from set Object.
=>If the Specified Value does not exist in set Object then we get KeyError.
--------------------------
Examples
--------------------------
>>> s1={10,"RS",34.56,True,2+3j}
>>> print(s1,type(s1),id(s1))
{'RS', True, 34.56, 10, (2+3j)} <class 'set'> 2378889695008
>>> s1.remove(10)
>>> print(s1,type(s1),id(s1))
{'RS', True, 34.56, (2+3j)} <class 'set'> 2378889695008
>>> s1.remove(34.56)
>>> print(s1,type(s1),id(s1))
{'RS', True, (2+3j)} <class 'set'> 2378889695008
>>> s1.remove(10)---------------------KeyError: 10
---------------------------------------------------------------------------------------------------------------------------------------------------------
3. discard()
---------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:      SetObj.discard(Value)
=>This Function removes the  Specified Value from set Object.
=>If the Specified Value does not exist in set Object then we get None OR Space as Result
---------------------------
Examples
---------------------------
>>> s1={10,"RS",34.56,True,2+3j}

			OR

------------------------------
>>> set().remove(10)---------------KeyError: 10

7. isdisjoint()
--------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:  SetObj1. isdisjoint(SetObj2)
=>This Function  returns True Provided There is No Common Element(s) between SetObj1 and SetObj2
=>This Function  returns False Provided There is atleast One Element(s) between SetObj1 and SetObj2
-----------------------------
Examples
-----------------------------
>>> s1={10,20,30,40}
>>> s2={20,25,35,45}
>>> s3={1,2,3,4}
>>> s1.isdisjoint(s3)
True
>>> s1.isdisjoint(s2)
False
>>> s3.isdisjoint(s2)
True
---------------------------------
>>> set("PYTHON").isdisjoint(set("Java"))
True
>>> set("PYTHON").isdisjoint(set("MANGO"))
False
----------------------------------
>>> set().isdisjoint(set())
True
>>> {10,20,30}.isdisjoint(set())
True

13. symmetric_difference()
--------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:    SetObj3=setobj1.symmetric_difference(setobj2)
=>This Function Removes the Common Elements from both SetObj1 and SetObj2 and take Remaining Elements from SetObj1 and SetObj2 and place those Elements in SetObj3.
--------------------------
Formula-1
--------------------------
 SetObj3=setobj1.symmetric_difference(setobj2) = setobj1.union(setobj2).difference(setobj1.intersection(setobj2))
 --------------------------
Formula-2
--------------------------
 SetObj3=setobj1.symmetric_difference(setobj2) = setobj1..difference(setobj2).union(setobj2.difference(setobj1))
------------------------
Examples
------------------------
>>> s1={10,20,30,40}
>>> s2={20,30,50,60}
>>>
>>> s3=s1.symmetric_difference(s2)
>>> print(s3,type(s3))
{40, 10, 50, 60} <class 'set'>
			OR
>>> s4=s1.union(s2).difference(s1.intersection(s2))
>>> print(s4,type(s4))------------{40, 10, 50, 60} <class 'set'>
			OR
>>> s5=s1.difference(s2).union(s2.difference(s1))
>>> print(s5,type(s5))------------{40, 10, 50, 60} <class 'set'>

>>> {10,20,30}.symmetric_difference({1,2,3})
