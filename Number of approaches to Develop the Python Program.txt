What is Program and its Purpose
----------------------------------------------------
=>A Program is a Collection of 'Optimized Instructions' Meant for Performing Certain Operation.
=>The 'Optimized Instructions' are having two Properties. 
				1. Less Time Complexity ( Amount of Time Taken by Processor to execute the program) 
				2. Less Space complexity ( Amount of Space Taken by the Program in main memory during execution)
=>In real time, after writing the Program, The Programmer Must save the Program on some File Name with an extension .py (File Name.py---Source Code).
NOTE: When we touch some thing on touch screen OR some thing u clicked on Browser then Internally Some Piece of Code (Program) will execute. 
---------------------------------------------------------------------------------------------------------------------------------------------------------------
=>In Real Time Python Program / Application can be developed in Two ways. They are
				1. By using Interactive Approach
				2. By using Batch Mode Approach
-------------------------------------------------------------------------------------------------------------------------------------------------------------
1. By using Interactive Approach
-------------------------------------------------------------------------------------------------------------------------------------------------------------
=>In Interactive Approach, The python Programmer Can Issue/Write One Instruction at a time and gets One output at a time.
=>The Interactive Approach is Useful To test One Instruction at a time.
=>The Interactive Approach is not Useful to Write group of Instructions for Solving Big Problems bcoz It is Not Possible Re-Use the Code in Other Part of the Project and more over It is not Permitting to save the code. To avoid this limitation, we must go for Batch Mode Approach.

Code in Python Command
--------------------------------------
>>> a=10
>>> b=20
>>> c=a+b
>>> print(a)---------------10
>>> print(b)---------------20
>>> print(c)---------------30

2. By using Batch  Mode Approach
-------------------------------------------------------------------------------------------------------------------------------------------------------------
=>In Batch  Mode Approach, The Python Programmer develops Group of Optmized Instructions for solving  any Problem 
    and saved on some file name with an extension .py(File Name.py----contains Source Code).
=>In real Time, we use always Batch  Mode Approach for solving any Problem  bcoz We can Re-Use the code any Part of 
    Project .
