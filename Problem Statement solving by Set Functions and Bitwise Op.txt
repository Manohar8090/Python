Consider the following Problem
		set of cricket players={"Rohit","Kohli,"Rossum"}
		set of tennis players={"Rossum","Travis","Hunter"}
Write the code for answring the following Questions

		1. Find all the player names who are playing all the Games
		2. Find all the player names who are playing Both Cricket and Tennis
		3. Find all the player names who are playing Only cricket But not tennis
		4. Find all the player names who are playing Only Tennis but not cricket
		5. Find all the player names who are playing eXclusively Cricket and Tennis.
----------------------------------------------------------------------------------------------------------------------------------------------------------------
				Solutions for above Problem
----------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Find all the player names who are playing all the Games--------union()
----------------------------------------------------------------------------------------------------------------------------------------------------------------
>>> cp={"Rohit","Kohli","Rossum"}
>>> tp={"Rossum","Travis","Hunter"}
>>> print(cp,type(cp))---------------{'Kohli', 'Rossum', 'Rohit'} <class 'set'>
>>> print(tp,type(tp))--------------{'Rossum', 'Travis', 'Hunter'} <class 'set'>
>>> allcptp=cp.union(tp)
>>> print(allcptp,type(allcptp))----------{'Travis', 'Kohli', 'Rossum', 'Rohit', 'Hunter'} <class 'set'>
----------------------------------------------------------------------------------------------------------------------------------------------------------------
2. Find all the player names who are playing Both Cricket and Tennis---intersection()
----------------------------------------------------------------------------------------------------------------------------------------------------------------
>>> cp={"Rohit","Kohli","Rossum"}

>>> print(tp,type(tp))-----------------{'Rossum', 'Travis', 'Hunter'} <class 'set'>
>>>bothcptp=cp.intersection(tp)
>>> print(bothcptp,type(bothcptp))-----{'Rossum'} <class 'set'>
----------------------------------------------------------------------------------------------------------------------------------------------------------------

Consider the following Problem
		set of cricket players={"Rohit","Kohli,"Rossum"}
		set of tennis players={"Rossum","Travis","Hunter"}
Write the code for answring the following Questions

		1. Find all the player names who are playing all the Games
		2. Find all the player names who are playing Both Cricket and Tennis
		3. Find all the player names who are playing Only cricket But not tennis
		4. Find all the player names who are playing Only Tennis but not cricket
		5. Find all the player names who are playing eXclusively Cricket and Tennis.
----------------------------------------------------------------------------------------------------------------------------------------------------------------
				Solutions for the above Problem
----------------------------------------------------------------------------------------------------------------------------------------------------------------
1. Find all the player names who are playing all the Games--------Bitwise  OR ( | )
----------------------------------------------------------------------------------------------------------------------------------------------------------------
>>> cp={"Rohit","Kohli","Rossum"}
>>> tp={"Rossum","Travis","Hunter"}
>>> print(cp,type(cp))---------------{'Kohli', 'Rossum', 'Rohit'} <class 'set'>
>>> print(tp,type(tp))--------------{'Rossum', 'Travis', 'Hunter'} <class 'set'>
>>> allcptp=cp|tp  #Bitwise  OR ( | )
>>> print(allcptp,type(allcptp))---------{'Travis', 'Kohli', 'Rossum', 'Rohit', 'Hunter'} <class 'set'>
----------------------------------------------------------------------------------------------------------------------------------------------------------------
2. Find all the player names who are playing Both Cricket and Tennis---Bitwise AND ( & )
----------------------------------------------------------------------------------------------------------------------------------------------------------------
>>> cp={"Rohit","Kohli","Rossum"}
>>> tp={"Rossum","Travis","Hunter"}
>>> print(cp,type(cp))----------------{'Kohli', 'Rossum', 'Rohit'} <class 'set'>
>>> print(tp,type(tp))-----------------{'Rossum', 'Travis', 'Hunter'} <class 'set'>
>>> bothcptp1=cp&tp  # Bitwise AND ( & )
>>> print(bothcptp1,type(bothcptp1))----------------{'Rossum'} <class 'set'>
----------------------------------------------------------------------------------------------------------------------------------------------------------------
3. Find all the player names who are playing Only cricket But not tennis------Arithmetic Minus Operator ( - )
----------------------------------------------------------------------------------------------------------------------------------------------------------------
>>> cp={"Rohit","Kohli","Rossum"}
>>> tp={"Rossum","Travis","Hunter"}
>>> print(cp,type(cp))---------------{'Kohli', 'Rossum', 'Rohit'} <class 'set'>

>>> print(onlycp,type(onlycp))-------------{'Kohli', 'Rohit'} <class 'set'>
----------------------------------------------------------------------------------------------------------------------------------------------------------------
4. Find all the player names who are playing Only Tennis but not cricket-------Arithmetic Minus Operator ( - )

>>> tp={"Rossum","Travis","Hunter"}

5. Find all the player names who are playing eXclusively Cricket and Tennis.------Bitwise XOR (^)
